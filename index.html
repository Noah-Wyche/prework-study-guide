HTML

  • the head element contains information about the webpage
  • the body element represents the visual element shown to the user
  • HTML is responsible for the structure of a webpage
  • There are many hidden from user view elements in HTML
  • The title element determines the name of the website in the browser

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A class can be added to any combination of elements
  • A class is designated with a period before its name in the style sheet
  • You can change colors and background colors
  • You can align text in a number of ways

Git

  • Git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • There is a specific workflow through Git to save your current project
  • First use the command "git add -A" to state our changes
  • Next use "git commit -m ''" to commit the changes locally
  • Make sure our local branch is up to date before pushing up our changes y running the following Git command: "git pull origin main"
  • To complete the Git flow, use "git push origin 'name of the branch'" to push changes to the remote repository
